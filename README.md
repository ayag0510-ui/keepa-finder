# Keepaプロダクトファインダー簡易版

Keepaの製品ファインダーの「よく使う項目だけ」に絞ったカンタン版ツールです。
リピート商材の横展開リサーチの入り口として使ってください。

## 使い方

1. [ツールを開く](index.html)
2. keepa.com にログイン →「API」ページで表示されているAPIキーをコピー
3. ツールの①にキーを貼り付けて「保存」
4. 条件を入力して「検索する」

APIキーはあなたのブラウザの中にだけ保存され、外部に送信されることはありません。
検索結果のASINは「全ASINをコピー」または「CSVダウンロード」でリピートリストに活用できます。
